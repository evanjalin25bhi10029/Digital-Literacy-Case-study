# Digital Literacy Case Study

## Introduction
Digital literacy is the ability to use digital tools and the internet effectively, safely, and responsibly.

## Importance
- Helps students access information
- Improves communication skills
- Supports online learning
- 
## Digital Literacy Case Study
A student uses digital tools like Google, YouTube, and online classes to learn new topics. They check whether the information is correct before using it.

They avoid unsafe websites and protect their personal information while using the internet.

This shows the importance of digital literacy in education.
